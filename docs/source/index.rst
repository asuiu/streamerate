Welcome to Streamerate's documentation!
===================================

**`Streamerate <https://github.com/asuiu/streamerate>`_** (/striːməˈreɪt/) is a powerful pure-Python library inspired by **`Fluent Interface pattern <https://en.wikipedia.org/wiki/Fluent_interface>`_** (used by Java 8 streams), providing a chainable and expressive approach to processing iterable data.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
